# eslint-config-tjw-jest

The Jared Wilcurt's Jest ESLint rules


## Using this

1. `npm install --save-dev eslint-config-tjw-jest`
1. In your `.eslitrc.js` add `tjw-jest` to your `extends` like so:
    ```js
    module.exports = {
      'extends': [
        'tjw-jest'
      ]
    };
    ```
